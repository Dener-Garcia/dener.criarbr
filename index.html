O que faço como um...

UX Designer

Através de processos e estudo do compotamento do usuário podemos aplicar a melhor solução para criação de um produto ou na melhoria de produtos existentes. Algumas ferramentas que utilizo são: Matrix CSD, Pesquisa, Definição de Personas, na definição constuimos UserFlow, UserJourney, Sitemaps e Wireframes, testes de usabilidade.

UI Designer

Aqui está uma paixão, desenvolver layouts que sejam atrativos, responsivos e com alta acessibilidade pensando com empatia e inclusão para abrangir os mais diversos grupos de usuários que possam vir a utilizar os produtos que desenvolvo. Utilizando o Figma podemos construir layouts, design system, prototipação de alta definição e trabalhar integrado com o desenvolvimento de produtos digitais.

Front end Developeder

Trazer ao mundo um projeto que impacte as pessoas de forma positiva, no desenvolvimento para a web moderna toda aplicação deve ser pensada para ser performatica e ao mesmo tempo acessivel para usuários que precisem de alguma ferramenta assistiva. Para desenvolver essas aplicações utilizo desde HTML, CSS, JavaScript até React e React Native.

Criador de conteudo

Ensinar para aprender, compatilho conhecimento através de redes sociais e do próprio Youtube afim de estar perto do mundo de design e desenvolvimento, conhecendo pessoas novas e contribuindo para a comunidade de alguma forma.
