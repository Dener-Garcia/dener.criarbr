Uma plataforma acessível para pessoas com autismo comprovar seu diagnóstico

Objetivo do projeto

No Brasil não existe uma plataforma, site ou aplicativo oficial para que pessoas com autismo comprovem seu diagnóstico, a ideia do Autista Live é justamente trazer essa facilidade para essas pessoas através de uma interface simples, rápida e eficiente.

Kick off

UX Process

Após a definição do projeto onde os dois principais pontos são a usabilidade para o usuário e um painel administrativo para aprovação das carteirinhas, parti para a criação do layout usando técnicas e heurísticas de design. Como temos diferentes níves de autismo a plataforma tem o objetivo de ser altamente intuitiva e fácil de usar.

Design

Ideação e prototipação

A inteface tem cores sólidas, os componentes de interação são bem acentuados afim de direcionar o usuário pelo caminho, as cores utilizadas como o azul para links e botões foi pensada para aproveitar o conhecimento que as pessoas já possuem sobre essas cores em elementos de interação.

Handoff

Ready for code

A integração entre design e código nem sempre é fluida já que ambos trabalham ao mesmo tempo no mesmo projeto, ter bem definido o tipo de entrega, componentização do projeto e uma boa comunicação visual é fundamental para a dead line em qualquer projeto de sucesso.

Desafios

Aprendizados

Projetar interfaces para abrangir um alto número de usuários é sempre desafiador, nesse projeto explorei como pessoas diversas podem utilizar um mesmo produto de maneira intuitiva.
