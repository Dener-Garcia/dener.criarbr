An accessible platform for people with autism to verify their diagnosis

Project Objective

In Brazil, there is no official platform, website, or application for people with autism to verify their diagnosis. The idea behind Autista Live is precisely to bring this convenience to these individuals through a simple, fast, and efficient interface.

Kick off

UX Process

After defining the project, where the two main points are usability for the user and an administrative panel for approving the cards, I proceeded to create the layout using design techniques and heuristics. Since we have different levels of autism, the platform aims to be highly intuitive and easy to use.

Design

Ideation and Prototyping

The interface features solid colors, with interaction components well emphasized to guide the user along the path. The colors used, such as blue for links and buttons, were chosen to leverage the existing user familiarity with these colors in interactive elements.

Handoff

Ready for Code

The integration between design and code is not always smooth since both work simultaneously on the same project. Having a well-defined delivery type, project componentization, and good visual communication is essential for meeting deadlines in any successful project.

Challenges

Learnings

Designing interfaces to cater to a large number of users is always challenging. In this project, I explored how diverse individuals can use the same product in an intuitive way.
